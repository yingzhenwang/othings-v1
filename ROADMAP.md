## 🚀 v1.1.0 - 智能增强版

### 目标

引入 LLM 能力，提升搜索体验；增加数据报表和导入导出功能。

### 功能清单

| # | 功能 | 描述 | 状态 |
|---|------|------|------|
| 2.1 | LLM 语义搜索 | 自然语言理解，智能匹配 | ⏳ |
| 2.2 | 资产总览 | 物品总数、总价值统计 | ⏳ |
| 2.3 | 分类分布 | 饼图/柱状图展示 | ⏳ |
| 2.4 | 提醒统计 | 即将到期/已过期统计 | ⏳ |
| 2.5 | 数据导出 | JSON 格式完整导出 | ⏳ |
| 2.6 | 数据导入 | JSON 格式导入 | ⏳ |
| 2.7 | 导入预览 | 预览变更，确认后执行 | ⏳ |

### 技术要点

```typescript
// LLM 搜索接口设计
interface LLMSearchOptions {
  query: string;           // 自然语言查询
  filters?: ItemFilters;   // 筛选条件
  limit?: number;          // 返回数量限制
}

interface SearchResult {
  items: Item[];
  relevance: number;       // 相关度 0-1
  summary?: string;        // 结果摘要（LLM 生成）
}
```

### 交付标准

- [ ] LLM 搜索能理解"我的电子产品"、"很贵的东西"等语义
- [ ] 报表数据准确，图表美观
- [ ] 导入导出数据完整性校验
- [ ] 导入预览清晰，冲突可处理

## 🔮 v2.0.0 - 同步与协作版

### 目标

支持多设备同步，增加高级功能，为未来扩展打下基础。

### 功能清单

| # | 功能 | 描述 | 状态 |
|---|------|------|------|
| 3.1 | 设备同步 | 可选云同步（WebDAFS/自托管） | ⏳ |
| 3.2 | 高级筛选 | 多维度组合筛选 | ⏳ |
| 3.3 | 批量操作 | 批量编辑/移动/删除 | ⏳ |
| 3.4 | 条码扫描 | 扫码添加物品（可选） | ⏳ |
| 3.5 | 模板功能 | 快速创建物品模板 | ⏳ |
| 3.6 | 数据可视化 | 价值趋势、保修日历 | ⏳ |

### 技术架构

